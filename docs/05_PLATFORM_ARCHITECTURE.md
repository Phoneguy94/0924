# Veytec Experience Platform Architecture

## Vision
Build a reusable platform that powers AI-driven event and customer experiences using shared components rather than custom projects.

## Core Principles
- Reusable components
- Configuration over customization
- Secure by design
- Professional user experience
- Fast deployment

## Shared Components
- Camera
- Gallery
- Wall
- Prompt Engine
- QR/Claim Flow
- Admin

## Experience Flow
Camera / Upload
→ Webex Connect
→ Airtable
→ AI Generation
→ Gallery
→ Wall

## Configuration
Each experience should be defined by configuration rather than HTML edits.

Examples:
- Branding
- Background
- Prompt Set
- Webhook
- Airtable mappings
- Gallery layout
- Wall layout

## Long-Term Goal
Launch a new branded experience in under 30 minutes without modifying shared platform code.
