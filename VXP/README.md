# VXP

Veytec Experience Platform

Status: prototype
Version: 0.1.0

VXP is the product folder for the reusable experience platform.

The current goal is to move from one-off event pages to a configurable platform with shared modules.

Core ideas:
- One platform
- Many experiences
- Shared components
- Configuration-driven events
- Secure-by-design architecture
- Reusable Webex and Airtable patterns

Initial folders:
- docs
- src
- config
- assets
- templates
- examples
- projects
