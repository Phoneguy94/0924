# VXP Project Vision

VXP is the Veytec Experience Platform.

The goal is to turn one-off event demos into a reusable platform for customer experiences.

## Use Cases
- Customer appreciation events
- Trade show booths
- Executive meetings
- Internal showcases
- Sports hospitality
- Migration excitement campaigns

## Product Direction
New experiences should be added through configuration whenever possible.

The platform should reuse camera, display, QR, gallery, Airtable, and Webex Connect patterns.

## Success
VXP succeeds when a new event can be launched quickly without copying and rewriting large HTML files.
